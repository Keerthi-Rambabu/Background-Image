Background Image

When we attempt to set an image as a background to occupy the full screen using HTML and CSS, it does cover the entire screen, but often, most of the background image gets cut off, not appearing as we intended (out of shape).

To address this issue, we can use the following CSS properties:

body{
margin:10px;
background-image: url("background.jpg"); <-- address or link of your image -->
background-size:100% 100%;
background-repeat: no-repeat;
background-position: cover;
}
